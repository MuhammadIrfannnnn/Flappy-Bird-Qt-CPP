# Flappy Bird Clone (C++ & Qt) 🚀

A cross-platform desktop implementation of the classic Flappy Bird game, developed using **C++** and the **Qt Framework**. This project focuses on object-oriented design, real-time collision detection, and efficient frame rendering.

---

### 🛠️ Technical Highlights
* **Engine:** Built using Qt's graphics framework for high-performance 2D rendering.
* **Game Loop:** Implemented robust timer events to maintain consistent frame rates.
* **Physics:** Custom gravity and velocity algorithms to simulate smooth bird "flapping" and falling.
* **Collision Detection:** Boundary checking between the bird sprite and dynamic pillar obstacles.

---

### 📂 Project Structure
- `main.cpp` / `mainwindow.cpp`: Entry point and main application window setup.
- `bird.cpp` / `bird.h`: Manages bird physics, state, and rendering.
- `pillar.cpp` / `pillar.h`: Logic for obstacle generation, movement, and recycling.
- `background.cpp` / `background.h`: Handles the scrolling background effect.
- `homescreen.cpp` / `homescreen.h`: UI state for the start menu.
- `hello.pro`: The main Qt Project configuration file.
